# UI Component Library & Design System

A reusable React.js component library built with SASS, Storybook, and Jest — following atomic design principles for scalable UI development.

## 🚀 Tech Stack
- React.js (Component-Based Architecture)
- SASS (Design Tokens — spacing, typography, color palette)
- Storybook (Live component documentation)
- Jest + React Testing Library (TDD, 90%+ coverage)
- ESLint + Prettier (Code quality enforcement)
- Rollup (Bundler)

## ✨ Features
- Reusable components: Buttons, Modals, Forms, Nav, Cards, Data Tables
- Responsive design with CSS Grid and Flexbox
- Design tokens for visual consistency across components
- Unit and integration tests for all components
- Live Storybook documentation with interactive demos

## 📦 Getting Started
```bash
npm install
npm run storybook
npm run test
```

## 🧪 Testing
```bash
npm run test --coverage
```
